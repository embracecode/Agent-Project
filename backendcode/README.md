## 🚀 使用指南

1. **准备配置文件**
   - 在 `config/` 目录下创建 `default.js`。
   - 填写你的阿里云 `apiKey` 等敏感信息，遵循上方示例。

2. **安装依赖并运行**
   ```bash
   npm install
   # 或者 yarn
   npm start
   ```

3. **开发与调试**
   - 控制器代码在 `controller/` 下。
   - 工具调用在 `config/tools.js` 和 `config/toolsFunctions.js` 中。

## 🛠️ 说明

- 聊天接口由 `controller/chat.js` 提供，依赖于配置中的系统提示和工具。
- 所有输入都通过 `validate` 目录中的函数进行基本检查。
- 公共格式化工具在 `utils/formatter.js`。

## 🔐 安全提示

- 切勿将真实的 `config/default.js` 提交到版本控制系统。
- 对于敏感的 API 密钥或凭证，使用环境变量或安全存储。
